  INSTALANDO O ALLEGRO NO MICROSOFT VISUAL C++ 2008 EXPRESS EDITION


Atualmente a biblioteca Allegro está na versão 4.2.2 e pode ser configurada para qualquer tipo de compilador e em muitas plataformas. Nesse artigo estarei ensinando a instalar o Allegro no compilador Microsoft Visual C++ 2008.

Na medida do possível estarei acompanhando as versões mais novas da biblioteca e atualizando este documento. Se você preferir pode pegar a documentação em inglês no site oficial do Allegro. O documento original é somente texto.

Para facilitar o processo estarei exemplificando esse documento com imagens.

Com o Visual C++ 2008 instalado no computador baixe os arquivos binários zipados no site do allegro.
Atualmente o nome é Microsoft Visual C++ 8.0.zip.

Link para download:
Pacote Allegro para Visual C++

Após carregar o arquivo no computador prossiga com as instruções abaixo para a configuração do Visual C++.

1) Após baixar o arquivo binário você deverá descompactar ele em uma pasta qualquer.
O resultado da extração dos arquivos na pasta deve ser conforme abaixo:



2) Se você fez a instalação padrão do Microsoft Visual C++ 2008 Express Edition ele deve estar instalado na pasta C:\Arquivos de programas\Microsoft Visual Studio 9.0\VC. Caso contrário você deve saber onde se encontra a pasta do VC para poder prosseguir com os passos abaixo.

Em uma janela localize a pasta onde foi instalado o VC++ e nao outra janela localize a pasta onde você extraiu os arquivos binários referente a biblioteca Allegro.

    1. Copie todo o conteúdo da pasta Include dos arquivos binários e cole dentro da pasta include do VC++. Visualize
    2. Copie todo o conteúdo da pasta lib dos arquivos binários e cole dentro da pasta lib do VC++. Visualize
    3. Copie todo o conteúdo da pasta bin dos arquivos binários e cole na pasta c:\Windows\System32. Visualize
    4. Copie apenas o arquivo alld42.pdb que se encontra dentro da pasta lib dos arquivos binários e cole na pasta c:\windows\system32. Visualize
Após fazer os passos acima estamos prontos para iniciar um projeto no VC++.


INICIANDO E CONFIGURANDO UM NOVO PROJETO

Após instalar o Allegro com os passos mencionados acima é necessário fazer algumas configurações para cada jogo que iremos criar.
Segue abaixo os passos para efetuarmos as configurações necessárias.

Criando um projeto no Visual C++:

  1. Abra o VC++ 2008, Clique no menu File->New->Project
  2. Selecione o tipo do projeto como General em seguida selecione o Empty Project. Coloque o nome do projeto e da pasta se você quiser criar uma.
  3. Após o VC++ criar o projeto vá para o menu e clique no botão add new item.
  4. Selecione a categoria Code em seguida o C++ file (.cpp). Chame esse arquivo de main e clique em Add

Se você fez tudo certinho até agora então você já deve ter a biblioteca Allegro instalada e um projeto novo iniciado.
Agora precisamos fazer com que o nosso projeto encontre a biblioteca Allegro.

Se não fizermos essa parte ao incluirmos a biblioteca no projeto o Visual C++ vai mostrar uma mensagem de erro dizendo que não encontra a biblioteca especificada.

Para a nossa sorte não precisamos mais fazer todas aquelas configurações no projeto que faziamos na versão do Visual C++ 2005.
A maioria das configurações que precisavamos fazer, nessa versão, já vieram por padrão da Microsoft.

É necessário apenas mostrar ao nosso projeto que ele é dependente do arquivo alld.lib para encontrar a biblioteca Allegro.
Faça conforme abaixo:

1) No seu projeto Clique no menu Project-> [Nome do projeto] Property. A tela abaixo irá aparecer.
Aparentemente o campo Additional Dependencies vai estar em branco.

Digite na linha em branco o texto: alld.lib, em seguida clique em OK para confirmar a alteração.

 

Feito todos os passos acima o seu projeto já deve estar configurado para encontrar e utilizar a biblioteca Allegro no compilador Visual C++ Express Edition 2008.




TESTANDO A INSTALAÇÃO

A seguir vou ilustrar a maneira mais simples de testar a instalação da biblioteca Allegro no Visual C++.
Copie e cole o código abaixo no seu projeto.

CÓDIGO...
//Código de Exemplo
#include <allegro.h>

int main()
{
   allegro_init();
   set_color_depth(16);
   set_gfx_mode(GFX_AUTODETECT_WINDOWED, 640, 480, 0, 0);
   allegro_message("Allegro: Alo Mundo");

   allegro_exit();
   return 0;
}
END_OF_MAIN();

FIM DE CÓDIGO

1) Para compilar o código pressione: Ctrl + F7
2) Para executar o programa pressione: F5





Caro leitor, chegamos ao fim do procedimento de instalação da biblioteca Allegro 4.2.2 no compilador Visual C++ Express Edition.

Se você conseguiu fazer os passos e estiver vendo a tela acima PARABÉNS, a instalação foi efetuada com sucesso.
Se houve algum problema durante os procedimentos citados acima tire suas dúvidas no fórum.
