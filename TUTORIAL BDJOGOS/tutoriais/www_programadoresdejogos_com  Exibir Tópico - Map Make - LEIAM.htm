E Gostaria de entrar ajudando o pessoal que racha a cuca para criar uma aplicacao em Delphi que crie mapas e os salve em um arquivo .seilá .

Primeiramente temos que criar uma nova aplicacão e colocar variaveis nela:

Código:

  public
        Quadro              :     Array[0..399] of TTiles;
        imagens             :     array[0..2] of string;
        dir                 :     String;
        box1,box2           :     TStringList;
        PisoSelecionado     :     integer;
        Procedure ClickQuadro(Sender:TObject);


Explicacões :

O Array quadro foi criado para abrigar as Tiles, a classe Tiles foi criada simplesmente erdando da classe Timage e adicionando a variavel Piso : String que é fundamental no save e load;
O Array imagens foi criado para abrigar o caminho das Imagens das Tiles;
A variavel dir é apenas usada para que quando futuramente carregaremos um mapa ( ao carregar um mapa, as imagens que compoe ele tem que estar no seu diretorio mas com as atribuicões feitas futuramente a ela nao terá que ter essas imagens no seu diretorio )
A Procedure ClickQuadro é criada para as Tiles do Array quadro adiquirirem uma certa imagem que será retirada do Array Piso posteriormente;
As StringLists box1 e box2 sao usadas para salvar o mapa;
A variavel PisoSelecionado é usada para pintar cada Tile independentemente;

Ok, Agora vamos atribuir os valores as variaveis, coloque o codigo no OnCreate do formulario :

Código:

     box2 := TStringList.Create;
     box1 := tstringlist.create;
     dir := GetCurrentDir;
     imagens[0] := dir+'\img\nada.bmp';
     imagens[1] := dir+'\img\agua.bmp';
     imagens[2] := dir+'\img\mato.bmp';
     For i := 0 to 399 do
     begin
          Quadro[i] := TTiles.Create(Form1);
          quadro[i].Left := (I*32) mod 640;
          quadro[i].top  := trunc((i/20))*32;
          quadro[i].Width := 32;
          quadro[i].Height := 32;
          quadro[i].OnClick := ClickQuadro;
          quadro[i].Piso := '0';
     end;


Explicacões :
Foi criado um grande quadro com varios Tiles, o nosso mapa.
Crie 3 Timages do lado para servir de Tinta para podermos selecionar oq vamos pintar no mapa, cada um dos 3 Timages tera as tags mudadas para 1 , 2 (a 0 nao é carregada pois é simplesmente nada)para podermos colocar no OnClick de uma imagem e que valerá para todas.

Olhe o OnClick:

Código:

procedure TForm1.Image1Click(Sender: TObject);
begin
     PisoSelecionado := TImage(sender).Tag;
end;


Como já dito a variavel PisoSelecionado pinta cada uma por isso aqui , olhe :

Código:
procedure TForm1.ClickQuadro(Sender: TObject);
begin
      if PisoSelecionado = 0 then exit;
      TTiles(sender).Picture.LoadFromFile(imagens[PisoSelecionado]) ;
      TTiles(Sender).tag := PisoSelecionado;
      TTiles(Sender).piso :=  inttostr(PisoSelecionado);

end;


A Procedure ClickQuadro dita acima, serve para pintar uma tile dito na explicaçao 1, ela é executada ao clicar em cima de um certo Tile.
