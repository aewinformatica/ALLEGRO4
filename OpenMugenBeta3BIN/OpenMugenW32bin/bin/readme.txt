OpenMugen PreAlpha Demo
-----------------------

This is the OpenMugen PreAlpha Demo.
It is the current compiled Win32 and linux bin 
and shows you the progress of OpenMUgen.

What is new?
-----------
Added TargetBind,TargetState, SelfState, Turn , TargetFacing controller
Added IsBound Trigger
Added Costume change states in hitdef controller
Collision detection is now better
Fixed Successive direction symbol reading and interpreting bug
Added GuardHandling
Added BindObject class for binding controllers
Added Targets of the player 

KFM can perform all his attacks now!!!!!!!!!!
