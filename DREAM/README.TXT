2. 关于按键

    游戏的按键模仿了游戏手柄的方向+四个功能键的模式.

    X  -----------  对话，调查
    D  -----------  弹出菜单，不过既然没有战斗，也就没什么价值了
    Z  -----------  返回，在操作菜单时用
    S  -----------  在小游戏中有特殊的功能
    F1 -----------  如果你不慎使DOS窗口冻结而造成黑屏的话，就按他吧
    F2 -----------  你可以用此键在游戏中截图，图片会被保存在Temp.bmp中
    数字键--------  做选择题时用到，是大键盘的数字键，小键盘的不管用
    方向键--------  似乎只有即时战略游戏是不用方向键的

3. 关于游戏

    由于我第一次写这么大的程序，很有一点手忙脚乱，顾此失彼。代码看上去乱了点，

剧情幼稚了点，美工也用了很多金点的。但是总的来说期望的效果还是达到了，至少在引

擎上做到了自编。

    说到引擎，还是参照了不少金点的经验。不过我打包票，我没翻看他一行代码(他那是

Visual C++的，看也看不懂)。其实金点的引擎也存在一些不足，脚本文件太多就是一个问

题，结果我也做成了这样。如果以后我在做引擎一定注意。

    在画地图的时候，我犯了一个RPG初编者都易犯的错误，就是地图画的太大了。看看《劫掠

轩辕剑》，《剑侠情缘外传》，地图都空旷的吓人，我的也是一样。(呵呵两位作者不要生气哦)

    压缩包中附带了引擎代码和地图编辑器。代码是纯C写的，本来是用TC2.0写的，后来移植

到DJGPP下，如果有人想看TC代码的话我的主页可以下载到。

    如果大家有什么看法或建议的话不妨发到我的邮箱(cattom1985@sina.com)，但请不要用粗

口，要砸的话轻点砸，呵呵。

    3月15日添加：今天将游戏移植到了Windows下，运行状况还好，不知其他机器上如何。不过

从此应该不会在有人说‘怎么没有声音’或‘我怎么黑屏’了  :)
