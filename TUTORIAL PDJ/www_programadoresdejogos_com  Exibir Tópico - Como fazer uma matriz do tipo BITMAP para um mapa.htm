Olá Nomulo

Fazer uma matriz de BITMAPs ocuparia muito espaço porque existem tiles repetidos e você os guardaria na memória várias vezes.
O jeito é criar um vetor de BITMAPs, como você fez, sendo cada um com um tile diferente.
Então você cria uma matriz de interios. O número que cada posição guardar será o índice do tile que você quer desenhar no lugar. Por exemplo, o tile01 é grama, o 02 é terra e o 03 água.
Então um mapa poderia ser
Código:
int mapa[4][4]=
{
  {1, 1, 1, 1},
  {1, 1, 2, 2},
  {1, 2, 2, 3},
  {2, 2, 3, 3}
}
E na hora de desenhar o mapa faz algo como
Código:
for(x=0;x<4;x++){
  for(y=0;y<4;y++){
    draw_sprite(screen, tile[mapa[y][x]], x*32, y*32);
  }
}
