Olá!

Estou fazendo um programa que desenha um mapa a partir de um arquivo texto. Por enquanto, apenas a idéia.

Acontece que, às vezes, o mapa não é desenhado, quando construo um mapa muito grande.

O código é o seguinte:

Código:

/* Lê o mapa de um arquivo texto e o desenha na tela */
int DesenhaMapa(char *mapa, BITMAP *tela){

   /* Ponteiro para o arquivo do mapa */
   FILE *arqMapa;
   
   /* Tipo de objeto que será desenhado */
   int tipo;
   
   /* Posição do objeto */
   int x = 0, y = 0;
   
   /* Cores */
   int branco = makecol(255,255,255);
   int vermelho = makecol(255,0,0);
   
   /*--------------------------------------------- INÍCIO ------------------------------------------------------*/
   
   /* Abre o arquivo para leitura */
   arqMapa = fopen(mapa,"rt");
   
   /* Se o arquivo não existe */
   if (arqMapa == NULL){
      return 0;
   };
   
   /* Lê o arquivo e desenha na tela */
   while (!feof(arqMapa)|| y >= SCREEN_H){
      fscanf (arqMapa, "%d\n", &tipo);
      switch(tipo){
          case 1:
             DesenhaQuadrado(tela, x, y, branco);
          break;

          case 2:
             DesenhaQuadrado(tela, x, y, vermelho);
          break;
      };
     
      /* Atualiza a posição horizontal do próximo objeto */
      x += LADO;
           
      /* Se o próximo objeto sair da tela */
      if (x >= SCREEN_W){
         
         /* Volta para o começo */
         x = 0;
         
         /* Vai para a próxima linha */
         y += LADO;
      };
   };
   
   /* Joga o mapa para a tela */
   draw_sprite(screen, tela, 0, 0);   
   
   /* Se deu tudo certo, fecha o arquivo e sai */
   fclose(arqMapa);
   return 1;
   
};


O mapa é construído da seguinte forma:

1) Crie um arquivo chamado mapa001.txt;

2) Dentro dele, coloque números, com espaços entre eles:
1-> Desenha um quadrado branco;
2-> Desenha um quadrado vermelho;
Outros-> Não desenha.

3) Você pode colocar um ao lado do outro, se quiser, mas as dimensões do mapa são de 32 X 24 quadrados (cada quadrado tem lado 20 e a reso-
lução de tela é de 640 X 480);
